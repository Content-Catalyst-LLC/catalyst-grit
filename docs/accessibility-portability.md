# Accessibility and Portable Operation

Catalyst Grit v2.0.0 keeps the public demo non-persistent and supports authenticated private workspaces, keyboard-operable workflow navigation, labelled regions, visible focus, live status announcements, responsive layouts, and reduced-motion preferences.

Portable snapshots use `catalyst-grit-portable-platform/2.0`. They include the private workspace export, connected workflow state, artifact graph, sync history, recovery instructions, and a deterministic SHA-256 bundle hash. Verification and restore do not require network access.
