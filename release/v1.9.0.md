# Catalyst Grit v1.9.0

**Publication, Export, API, and Institutional Integration**

This release makes Catalyst Grit portable and governable across institutional workflows. It adds seven report types, JSON/JSON-LD/Markdown/HTML/CSV exports, platform PDF requests, deterministic publication bundles, an authenticated and scoped API contract, rate limiting and append-only request audits, institutional policy versions, access reviews, methodology governance, schema deprecation guidance, diagnostics, and migration 008.

Catalyst Grit remains private by default and prohibits diagnosis, employee ranking, hidden performance evaluation, and automated eligibility decisions.
