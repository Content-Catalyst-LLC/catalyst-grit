# Catalyst Grit Methodology

Catalyst Grit treats recovery as a human-systems condition shaped by impact, pressure, constraints, support, available capacity, response readiness, learning, and bounded next steps.

```text
context → trigger → impact → pressure → constraints → supports → capacity → response → learning → next steps → human review
```

The engine keeps every component visible, supports versioned thresholds, and always records interpretation limits. Human review remains responsible for consequential use.
