# Catalyst Grit v1.4.0 — Recovery Planning and Action Management

v1.4.0 converts mapped recovery conditions into a structured, owned, and reviewable action plan.

## Release scope

- Rich canonical action contract and executable plan view
- Smallest recoverable next step and four planning horizons
- Dependencies, support needs, expected effects, effort, urgency, and evidence
- Scope decisions, blockers, escalation, checkpoints, and reassessment triggers
- Migration 003 with append-only action events and reassessments
- Action, blocker, and reassessment repository and CLI services
- Plan-history export and import support
- Browser and WordPress planning controls and summaries
- Backward compatibility through v1.3.0

## Safety boundary

Blocked actions and passed target dates identify support, scope, or sequencing needs. Catalyst Grit does not interpret them as individual failure or use them for employee evaluation, ranking, diagnosis, automated eligibility, or outcome prediction.
