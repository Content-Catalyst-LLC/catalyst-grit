# Monitoring, Trends, and Resilience Signals

Catalyst Grit v1.9.0 stores immutable monitoring snapshots linked to the exact recovery-record revision, source hash, engine version, schema version, methodology profile, and JSON source paths.

## Dashboards

- Personal/private record dashboard with traceable score and component history.
- Project conditions dashboard with action, blocker, checkpoint, pattern, and system-change summaries.
- Team/system conditions dashboard with a minimum consented group-size threshold of three.
- Recovery timeline joining revisions, snapshots, action events, checkpoints, and reassessments.

## Signal governance

Two snapshots are required before a directional trend is reported. Sparse data remains explicit. Historical calculations are never silently recalculated under a newer engine. Predictive claims, individual ranking, hidden scoring, diagnosis, and automated performance decisions are prohibited. Institutional interpretation requires an append-only human review.
