# Catalyst Grit WordPress v1.7.0

Shortcodes:

- `[catalyst_grit_demo]` — public, non-persistent browser recovery and learning demo.
- `[catalyst_grit_workspace]` — authenticated, per-user private workspace bundle storage.

The demo exposes keyboard-accessible controls and output panels for condition maps, recovery planning, retrospectives, adaptation patterns, evidence ledgers, assumption registers, cross-product handoffs, system-change guidance, and canonical JSON.

The plugin does not diagnose health, assign personality labels, score employee performance, or register anonymous persistence handlers.
