# Catalyst Grit v1.6.0

## Team Recovery and Facilitated Review

This release adds private team roles, consent-aware facilitated sessions, scoped perspectives, shared agreements, append-only agreement history, team export/import, CLI workflows, and an expanded authenticated WordPress workspace boundary.

The release does not introduce individual scoring, participant ranking, diagnosis, automated eligibility, or hidden employee evaluation.

### Database

Migration 005 adds the team recovery and facilitated review tables and creates owner memberships for existing projects.

### Compatibility

Canonical recovery records from v1.0.1 through v1.5.0 remain accepted. Existing SQLite workspace state is migrated in place and preserved by the installer.
