# Catalyst Grit WordPress v2.0.0

Shortcodes:

- `[catalyst_grit_demo]` — public, non-persistent recovery and learning demo.
- `[catalyst_grit_workspace]` — authenticated, per-user connected resilience workspace with workflow, portability, publication, evidence, monitoring, and governance guidance.

REST integration:

- `GET /wp-json/catalyst-grit/v1/health` and `/v2/health` — public non-sensitive integration status.
- `GET /wp-json/catalyst-grit/v1/workspace` and `/v2/workspace` — authenticated workspace retrieval.

The Python service supplies the production connected workflow, artifact graph, portable snapshots, institutional API, publication renderer, token hashing, policies, access reviews, and append-only audits.

The plugin does not diagnose health, infer personality, rank employees, perform automated eligibility, or register anonymous persistence handlers.
