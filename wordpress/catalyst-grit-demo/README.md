# Catalyst Grit WordPress v1.9.0

Shortcodes:

- `[catalyst_grit_demo]` — public, non-persistent browser recovery and learning demo.
- `[catalyst_grit_workspace]` — authenticated, per-user private recovery, publication, evidence, monitoring, and governance bundle storage.

REST integration:

- `GET /wp-json/catalyst-grit/v1/health` — public non-sensitive plugin integration status.
- `GET /wp-json/catalyst-grit/v1/workspace` — authenticated user workspace retrieval.

The production scoped institutional API, publication renderer, token hashing, rate limits, policies, access reviews, and append-only request audits are supplied by the installable Python service contract.

The plugin does not diagnose health, infer personality, rank employees, perform automated eligibility, or register anonymous persistence handlers.
