# Catalyst Grit v1.2.0

**Release:** Persistent Records, Projects, and Review Checkpoints  
**Date:** 2026-07-17

This release establishes the stable domain contract used by Python, CLI, browser, schemas, examples, tests, and API integrations.

## Release outcomes

- One nested recovery-record contract with strict lifecycle and provenance.
- Separate user input, normalized input, generated findings, and human review.
- Seven explainable weighted components under a versioned methodology profile.
- Structured validation errors and controlled namespaced extensions.
- Golden Python/JavaScript parity across standard, reviewed, high-friction, and custom-profile records.
- Migration of v1.0.x flat requests without retaining trait-oriented language.

## Compatibility

v1.0.x flat input is accepted by the engine and routed through the documented migration adapter. Generated output is always v1.2.0.

## Release gate

The release is valid only when all tests, schemas, examples, parity fixtures, syntax checks, package installation, portable smoke checks, and deterministic artifact checks pass.
