# Catalyst Grit v1.7.0

**Evidence, Assumptions, and Decision Handoffs**

This release adds traceable evidence and source records, explicit assumption lifecycle management, cross-product artifact handoffs, stale and conflict validation, a Decision Studio packet contract, migration 006, CLI workflows, project export/import, and an expanded authenticated WordPress workspace.

Catalyst Grit remains private by default and non-diagnostic. Evidence strength and assumption confidence describe the state of recorded information, not the character, health, or performance of a person.
