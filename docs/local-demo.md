# Local Demo

Run the Python example:

```bash
python3 python/catalyst_grit_core.py examples/grit_record_input.json --format json
python3 python/catalyst_grit_core.py examples/grit_record_input.json --format markdown
```

Run tests:

```bash
python3 -m pytest tests
```

The WordPress demo can be installed from `dist/catalyst-grit-demo.zip` after running the upgrade script.
