# Catalyst Grit v1.5.0 — Learning Loops and Adaptation Patterns

v1.5.0 adds a traceable learning layer to the canonical recovery record and private project workspace.

## Release outcomes

- Structured retrospectives preserve what happened, expectations, changes, evidence, and uncertainty.
- Explainable adaptation-pattern candidates cite exact source paths and values.
- Users can accept, reject, or correct candidates.
- Project workspaces aggregate recurring observations without personality labels.
- Source-linked system-change proposals support pilot windows and adopt/revise/defer/retire decisions.
- Retrospectives, pattern reviews, and system-change events are append-only.
- Exports preserve learning evidence, uncertainty, reviews, sources, and decision history.
- v1.0.x through v1.4.0 inputs and existing v1.4.0 SQLite workspaces remain compatible.

## Safety boundary

Pattern detection is descriptive and reviewable. It must not be used to diagnose health, score character, rank employees, infer personality, determine eligibility, or automate performance decisions.
